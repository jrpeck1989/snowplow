Building an Enriched event relay for GTM SS

This documentation is for Open Source pipeline installations which want to explore using Google Tag Manager Server Side with Enriched Events.

  1. Set up theÂ Snowplow ClientÂ within your GTM Server Side Container.
  2. Create aÂ Custom Integration application for AWS or GCP.
  3. Use anÂ Analytics SDKÂ to convert to an Enriched JSON.
  4. Send the JSON event as POST request toÂ https://{{gtm-ss-url}}/com.snowplowanalytics.snowplow/enriched.
    • The payload can be a single Enriched JSON event, or an array of Enriched JSON events.
